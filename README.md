# 📜 胚胎實驗室回報訊息小工具 (Embryo Message Helper)

這是一個專為試管嬰兒 (IVF) 實驗室設計的網頁工具，協助胚胎師快速生成標準化、格式統一的通知訊息，以便發送給個案或醫師。

🔗 **線上使用：** [https://zt0121.github.io/embryo_message/](https://zt0121.github.io/embryo_message/)

---

## ✨ 主要功能

### 1. 📜 報胚訊息 (Embryo Report)
* 針對 D1~D7 不同階段生成個案通知。
* **智慧邏輯**：自動推算查詢日、依據選項自動隱藏/顯示切片欄位。
* **特殊情境**：支援 D3 植入但 D5 無冷凍/無切片等特殊回報狀況。

### 2. 🧬 補救受精回報 (Rescue ICSI)
* 整合補救受精與正常受精顆數，自動計算總胚胎數與查詢日期。

### 3. 🧪 凍精訊息產生器 (Sperm Freezing)
* **雙向溝通**：分別產生給「醫師」與「個案」的專用訊息。
* **異常標記**：自動判斷精蟲數值（體積/濃度/活動力）並標註異常。

### 4. 📦 轉入｜輸出 回報 (Transfer/Export)
* **名冊管理**：內建醫師名冊功能，支援單筆或多筆批次與醫師回報。
